1.What will be the value of s, if n=127?

Read n

i=0,s=0

Function Sample(int n)

while(n>0)

r=n%10

p=8^i

s=s+p*r
i++
n=n/10
End While
Return s;
End Function
a.27

Explanation:
The above code is converting an octal number into its decimal representation. Here we are treating 127 as an octal input and converting it into its decimal representation, that is 87. So, answer will be 87.
